opahfirev

(Linux) Reports hardware and firmware information about the Intel® Omni-Path Host Fabric Interface (HFI)s.

Syntax

opahfirev

Options

None.

Example 

# opahfirev
######################                                       
server.intel.com  - HFI 02:00.0
Board: ChipABI 3.0, WFR_ID 0x1, ChipRev 7.0 patch 0x00000003, SW Compat 3
SN:    0x0057501a
Bus:   PCIe,8000MHz,x16
GUID:  0011:7501:0157:501a
