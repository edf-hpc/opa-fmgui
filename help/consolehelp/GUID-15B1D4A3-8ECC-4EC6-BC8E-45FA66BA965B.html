opafirmware

Returns firmware information.

Syntax

opafirmware --showVersion firmwareFile

or 

opafirmware --showType firmwareFile

Options

--help

Produces full help text.

--showVersion firmwarefile

firmwarefile is the file name.

--showType firmwarefile

firmwarefile is the file name.

Examples 

# opafirmware --showVersion STL1.q7.10.0.0.0.spkg
10.0.0.0
# opafirmware --showType STL1.q7.10.0.0.0.spkg
Omni_Path_Switch_Products.q7
