Routes

When multiple device nodes are selected, this panel shows the local topology of the selected nodes. It also highlights the routes among the nodes to show how they connect to each other.

Detailed Information Panel

When multiple device nodes are selected, this panel shows the pairwise route among the selected nodes in a tabbed pane. Each tab shows the route as a connectivity table (see Connectivity (Node) ). It shows the whole path from one port to another and each port's performance data.
