Links

When you select one or more ports, this panel shows and highlights each port's connection.

Detailed Information Panel

When multiple ports are selected, this panel shows each port's connectivity table in a tabbed pane. Use the More button on the top-right corner to display additional links. When the number of links is less than 5, the button is disabled.
