Virtual Fabric Management

This management tool is the same as Applications Management except that groups are replaced with virtual fabrics.

On the right panel, you can specify basic administrator policies, for example, whether to enable one virtual fabric and whether it should provide security control and/or QOS control.

The administrator policies attributes can be used only once and the used attribute is identified with an icon to indicate that it is not available.
