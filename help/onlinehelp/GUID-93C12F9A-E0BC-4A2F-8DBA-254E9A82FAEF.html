Admin Tab

The Admin tab provides tools to manage a subnet. The control panel on the left lists all available functions. The right side displays the corresponding content for a selected function.

The first time you select one of the icons on the control panel, you must enter a valid username and password to log in, which gives you access to the administrative features. After the initial login, admin features, there is no need to enter the credentials again.

The following functions are described in the linked topics:

  • Applications Management
  • Device Group Management
  • Virtual Fabric Management
  • Interactive Console
  • Log View
