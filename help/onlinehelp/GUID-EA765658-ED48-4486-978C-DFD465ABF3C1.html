Pin Board

The Fabric Manager GUI provides a pin board at the right side of the main window (see the following figure) to monitor cards of interest. Click the pin button on a card’s title bar to make it instantly visible on the pin board, which allows you to monitor performance metrics while exploring other parts of the system. For example, the congestion trend card can be pinned to the pin board and it remains visible even when looking at the performance of a specific port. To remove cards from the pin board, click the button.

Cards pinned to the Pin Board

The Event Summary is the default pin card that shows the summary of events in a defined time window; it cannot be unpinned. Clicking the Event Summary card toggles the event panel at the bottom of the main window. Below it are user pin cards that are created by clicking the pin button on a card's title bar. This board holds a maximum of eight pin cards. Each pin card is resizable by dragging the bottom edge. The following table summarizes the available operations on a pin card.
