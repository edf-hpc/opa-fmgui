Node Properties

The Node Properties section displays the properties for a selected node. The tool bar contains the following buttons:

  • Apply Options – select which properties to display
  • Show Border – turns on/off borders on property items
  • Uniform Rows – turns on/off alternative row colors

*Other names and brands may be claimed as the property of others.
