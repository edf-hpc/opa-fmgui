Device Node

When a device node is selected, this panel shows the connections of the node. The selected node is highlighted with a green border to distinguish it from its neighbors.

Detailed Information Panel

When a device node is selected this panel displays this node's properties and connectivity table, which is the same data represented with different views. For more detailed information see Connectivity (Node) and Node Performance.
